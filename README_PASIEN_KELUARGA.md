# Dokumentasi Sistem Data Keluarga Pasien

## Overview

Sistem ini dirancang untuk mengelola data keluarga pasien dan riwayat rekam medis mereka dengan antarmuka yang user-friendly dan responsif.

## Struktur Sistem

### 1. Model Database

- **Pasien**: Menyimpan informasi pribadi pasien
- **RekamMedis**: Menyimpan riwayat kunjungan dan rekam medis

### 2. Relasi Database

```php
// Model Pasien
- hasMany RekamMedis

// Model RekamMedis
- belongsTo Pasien
- belongsTo Dokter
- belongsTo Poli
```

### 3. Controller

- **KeluargaPasien**: Mengelola operasi CRUD untuk data keluarga

### 4. Halaman Frontend

- **Index.jsx**: Daftar semua anggota keluarga
- **Show.jsx**: Detail informasi pasien dan riwayat medis

## Fitur Utama

### Halaman Index

- ✅ Daftar anggota keluarga dalam bentuk kartu
- ✅ Pencarian berdasarkan nama, KTP, atau BPJS
- ✅ Navigasi cepat ke detail
- ✅ Responsive design

### Halaman Detail

- ✅ Informasi lengkap pasien
- ✅ Riwayat rekam medis dengan timeline
- ✅ Tab navigasi untuk informasi tambahan
- ✅ Desain modern dan user-friendly

## Cara Penggunaan

### 1. Menambahkan Data

```bash
# Akses halaman index
GET /pasien/keluarga

# Akses halaman detail
GET /pasien/keluarga/{id}
```

### 2. Pencarian

- Gunakan kolom pencarian di halaman index
- Pencarian berfungsi untuk nama, nomor KTP, dan nomor BPJS

### 3. Styling

- Menggunakan Tailwind CSS untuk styling modern
- Komponen UI dari shadcn/ui untuk konsistensi
- Responsive design untuk semua ukuran layar
